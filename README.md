# 2024 대구대학교 졸업작품 WhereDU
2024.05.11 ~ 2024.08.19<br>
2024 대구대학교 졸업 작품으로 무엇을 만들까, 고민하다가, 친구와 약속을 잡고 약속을 잊어버리거나 서로의 위치를 몰라 답답하여, 이런 앱이 있으면 좋을 것 같아서 만들게 되었다.<br>
